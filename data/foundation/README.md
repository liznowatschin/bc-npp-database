# Foundation Data

This directory contains reviewed, public-safe foundation artifacts for BC-NPPD
release preparation. Generated outputs, raw source files, private review notes,
and local materialization products do not belong here.

The `v1.0.0a/` directory is a small foundation scaffold, not a full database
release. It includes one reviewed example species record and sidecar files that
exercise the source-attribution, canonical-table, and scoring contracts.
