# Code Of Conduct

BC-NPPD follows the UBC FRESH Lab expectation that collaboration should be
respectful, constructive, and evidence-oriented.

Project participants should:

- Treat collaborators, students, practitioners, and public contributors with
  respect.
- Critique ideas, data quality, and implementation choices without personal
  attacks.
- Preserve uncertainty in ecological data rather than overstating confidence.
- Keep public repository spaces free of private, sensitive, or inappropriate
  material.

Concerns should be raised with the project maintainers.
